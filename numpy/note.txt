对于同样的数值计算任务，使用 numpy 比直接编写python 代码实现的 优点
1. 代码更加简洁   numpy 直接以数组 ，矩阵 为粒度计算并且支持大量的数学函数，
                 而 python 需要使用底层实现
2. 性能更高效     numpy 的数组存储效率和输入输出的性能都比 python 使用  list嵌套要好

注  numpy 的数据存储  和  python 是不一样的， 因为其底层是 使用  C 直接写的
